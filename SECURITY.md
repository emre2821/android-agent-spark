# Security Policy

## Reporting a vulnerability
If you discover a security issue, please email security@android-agent-spark.dev with details and steps to reproduce. You may also open a private security advisory on GitHub.

## Response expectations
- Acknowledge receipt within 2 business days.
- Initial assessment and mitigation plan within 7 business days.
- Coordinated disclosure timeline agreed with reporter before publication.

## Scope
All code, build pipelines, and deployment assets in this repository are in scope unless explicitly stated otherwise.

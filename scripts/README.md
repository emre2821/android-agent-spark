# Build Scripts

JavaScript build scripts for mobile and desktop packaging.

## Available Scripts

### `build_mobile.js`
Builds and stages web assets for the Capacitor/Android mobile wrapper.

```bash
node scripts/build_mobile.js
```

### `build_desktop.js`
Builds and stages web assets for the Electron desktop wrapper.

```bash
node scripts/build_desktop.js
```

## Usage

Run from the repository root:

```bash
npm run build:mobile
npm run build:desktop
```

# Shell Scripts

Shell scripts for building, deployment, and development automation.

## Available Scripts

### `build_apk_debug.sh`
Builds a debug APK for Android development and testing.

```bash
./scripts/shell/build_apk_debug.sh
```

### `run_on_termux.sh`
Script to run Agent Spark on Termux (Android terminal emulator).

```bash
./scripts/shell/run_on_termux.sh
```

### `new.sh`
Helper script for creating new components or features.

```bash
./scripts/shell/new.sh
```

## Usage

Make sure scripts are executable:

```bash
chmod +x scripts/shell/*.sh
```

Then run from the repository root:

```bash
./scripts/shell/<script-name>.sh
```

# TypeScript Common

Shared TypeScript/JavaScript utilities and types for Agent Spark applications.

## Overview

This package contains shared code that can be used across multiple applications:

- Common type definitions
- Utility functions
- Shared constants
- API helpers

## Usage

Import shared utilities in your application:

```typescript
import { someUtility } from '@agent-spark/ts-common';
```

## Development

```bash
# Install dependencies
npm install

# Build
npm run build

# Run tests
npm test
```
